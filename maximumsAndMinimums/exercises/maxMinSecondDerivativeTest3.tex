\begin{exercise}

The function $f(x) = x\ln(x)$ has only one critical number, $x=a$. 
$$
a = \answer{1/e}
$$


$f''(a)$ is \wordChoice{\choice[correct]{positive} \choice{negative}}, so $x=a$ is a local \wordChoice{\choice{max} \choice[correct]{min}}.
